# 💰 MoneyTrack

## 📖 Description
MoneyTrack est une application simple de suivi des dépenses et des revenus.  
Elle permet aux utilisateurs de mieux gérer leur budget et d’avoir une vision claire de leurs finances au quotidien.  

## ✨ Fonctionnalités
- ✅ Ajouter / modifier / supprimer une dépense ou un revenu  
- ✅ Visualisation des transactions sous forme de liste  
- ✅ Graphiques pour analyser les revenus et dépenses  
- ✅ Sauvegarde locale des données  

## 🛠️ Technologies utilisées
- JavaScript (Frontend)  
- Chart.js (Graphiques)  
- LocalStorage (Persistance des données)
